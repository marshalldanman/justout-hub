THE FINGER — Setup Guide
========================

Quick start:
  1. Set up Google service account (step 1 below)
  2. Create finger_config.json (step 2)
  3. Set up the Google Sheet (step 3)
  4. pip install -r requirements.txt
  5. python3 finger.py --dry-run


1. GOOGLE CLOUD SERVICE ACCOUNT
   - Go to console.cloud.google.com
   - Create a project (or use existing)
   - Enable: Google Sheets API + Google Drive API
   - IAM & Admin → Service Accounts → Create
   - Download the JSON key → save as service_account.json here
   - (Already in .gitignore — won't get committed)


2. CONFIG FILE (finger_config.json)

   Create this file in the repo root:

   {
     "service_account_file": "service_account.json",
     "spreadsheet": "AI Central Command",
     "email_to": "you@gmail.com",
     "email_from": "you@gmail.com",
     "email_password": "xxxx-xxxx-xxxx-xxxx",
     "smtp_server": "smtp.gmail.com",
     "smtp_port": 587,
     "warn_hours": 48
   }

   For Gmail app password:
   - Enable 2FA on Gmail
   - myaccount.google.com → Security → App Passwords → generate one

   (finger_config.json is in .gitignore — your secrets stay local)

   Alternatively, use env vars instead:
     GOOGLE_SA_KEY, FINGER_SHEET, FINGER_EMAIL_TO,
     FINGER_EMAIL_FROM, FINGER_EMAIL_PASS


3. GOOGLE SHEET SETUP
   - Create a Google Sheet named "AI Central Command"
   - Share it with the service account email (ends in @...iam.gserviceaccount.com)
   - Give it Editor access

   Create these tabs:

   Tab: "Tasks"
   ┌──────────────────────────┬────────────┬──────────┬────────┐
   │ Task                     │ Deadline   │ Priority │ Status │
   ├──────────────────────────┼────────────┼──────────┼────────┤
   │ File 2025 quarterly tax  │ 2026-04-15 │ high     │ active │
   │ Reconcile bookkeeping    │ 2026-03-20 │ high     │ active │
   │ Renew domain             │ 2026-06-01 │ med      │ active │
   └──────────────────────────┴────────────┴──────────┴────────┘

   Tab: "Commands"
   ┌──────────────────┬──────────────────────────────────────┬─────────┬────────┐
   │ Source           │ Text                                 │ Status  │ Result │
   ├──────────────────┼──────────────────────────────────────┼─────────┼────────┤
   │ Claude Code      │ [remind] Check on IRS letter         │ pending │        │
   │ ChatGPT          │ [urgent] Tax deadline moved to 4/10  │ pending │        │
   │ Brainstorm       │ [note] New app idea for bookkeeping  │ pending │        │
   └──────────────────┴──────────────────────────────────────┴─────────┴────────┘

   Add any other tabs you want (Brainstorm, Ledger, Bookkeeping 2025, etc.)
   Drop [brackets] anywhere — The Finger will find them.


4. USAGE

   python3 finger.py              # check everything, send emails
   python3 finger.py --dry-run    # show what WOULD happen, send nothing
   python3 finger.py --status     # print tasks & commands overview
   python3 finger.py --reset      # clear log, re-alert everything


5. AUTOMATE (cron)

   crontab -e
   */30 * * * * cd /path/to/justout-hub && python3 finger.py >> finger_cron.log 2>&1


BRACKET COMMANDS
================
Drop these anywhere in any cell in the Google Sheet:

  [remind]   — sends you a reminder email with surrounding text
  [urgent]   — sends a high-priority critical alert
  [task]     — flags something as a new task
  [email]    — forwards the text as an email
  [note]     — logs it quietly, no email
  [snooze]   — explicitly skip/ignore
  [anything] — any other bracket text sends a general notification

Examples:
  "Need to call accountant [remind] before Friday"
  "[urgent] IRS notice received, respond by March 20"
  "[task] Set up QuickBooks integration"
  "This might work for the API [note] research later"

The idea: you can be in ANY sheet — brainstorm, ledger, bookkeeping —
and just drop a [bracket] inline. The Finger picks it up next run.
